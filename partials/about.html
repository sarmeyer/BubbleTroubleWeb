About Bubble Trouble

Back

A simple, yet addictive game with the goal of popping as many bubbles as you can before the time runs out. Built with Unity, Xcode, and C#, Bubble Trouble is a free mobile app coming soon to the app store, but available to you now on the web.

Devin and Sarah are currently enrolled in the Web Development Immersive Program at Galvanize, Denver. They built this game to explore unfamiliar environments in software and game development. To learn more about the creators of Bubble Trouble, visit the links below.
